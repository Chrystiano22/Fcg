3. DESCRICAO RESUMIDA DA ENTREGA

O projeto entregue nesta fase implementa o MVP da plataforma FIAP Cloud Games em .NET 8.

Escopo coberto:

- cadastro de usuarios
- autenticacao com JWT
- autorizacao com perfis User e Administrator
- administracao de jogos
- biblioteca de jogos adquiridos
- promocoes administradas por perfil Administrator
- persistencia com Entity Framework Core e SQLite
- migrations
- Swagger
- logs estruturados
- testes automatizados
- documentacao DDD
